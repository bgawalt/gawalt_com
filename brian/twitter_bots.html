Brian's Twitter Bots

A bunch of bots I made for Twitter. They're all on hiatus right because I'm upset with how that website started running. In order of how much I like them:
  • TK: Each wordsquare

  • TK: ReaderBot

  • Synthetic Shakespeare: I trained a Markov model on the digram transitions of the works of Shakespeare, and now I can sample new Shakespeare-sounding verses. (July 2010)

  • TK: RandomBabel

  • 1-900-BAD-SEXT A bot that goes out and finds present-tense tweets, then repeats them after the famous "SEXT:" prefix. So long, context! Source code

  • Acrostician A bot that keeps a running count of how often n-grams appear on Twitter, then tries to patch together a reasonable sounding acrostic which matches a specified target. Source code
